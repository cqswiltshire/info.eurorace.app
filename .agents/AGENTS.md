# Project Rules

- Always save artifacts or implementation plans in the workspace directory (e.g., as `implementation_plan.md` in the project root) rather than the default hidden `.gemini` artifact folder, so the user can easily access and read them.
- Always ensure the styling and UI components of this marketing site remain visually consistent with the main application. Refer to `reference-eurorace/tailwind.config.ts`, `reference-eurorace/components.json`, and global CSS in the main app for design tokens, colors, fonts, and UI components.
- All spelling across the codebase (including user-facing text, documentation, and comments) should be British English.
